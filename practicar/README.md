# 自己紹介ページ

このリポジトリは、カラフルな背景と簡単な自己紹介情報を表示する1ページのウェブサイトです。

## 構成

- `index.html`: 自己紹介ページのHTMLファイル
- `styles.css`: ページのデザインを定義するCSSファイル

## セットアップ方法

1. このリポジトリをクローンまたはダウンロードしてください。
2. 必要に応じて、`index.html` と `styles.css` ファイルを編集してください。
3. ローカルマシンで `index.html` をウェブブラウザで開いてください。
