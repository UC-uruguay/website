【ステップ1】chatbot-proxy.php をアップロード
─────────────────────────────────
1. さくらのコントロールパネルにログイン
2. ファイルマネージャーを開く
3. WordPressのディレクトリに移動
4. このファイルをアップロード:
   /home/uc/uc-site/chatbot-proxy.php

確認: https://uc.x0.com/chatbot-proxy.php
→ {"status":"ok",...} と表示されればOK


【ステップ2】WordPressにログイン
─────────────────────────────────
https://uc.x0.com/wp-admin/

ユーザー名: uc-japan
パスワード: （WordPressのパスワード）


【ステップ3】WP Codeプラグインをインストール
─────────────────────────────────
左メニュー > プラグイン > 新規追加
検索: "WP Code"
インストール & 有効化


【ステップ4】スニペット追加
─────────────────────────────────
左メニュー > Code Snippets > Add Snippet
Code Type: HTML Snippet
タイトル: UC Chatbot

コード: wordpress-simple-embed.html の内容をコピペ
（圧縮版は「簡単セットアップ手順.md」に記載）

Location: Sitewide Footer
Active: ON
保存


【確認】
─────────────────────────────────
https://uc.x0.com/ にアクセス
→ 右下にチャットボタンが表示される

テスト:
- "UCは何歳ですか？" → 35歳
- "出身は？" → 福岡県
