## スマホとの同期方法（3つの選択肢）

### 方法1: Git 【推奨】

**メリット:**
- バージョン管理ができる
- どこからでもアクセス可能
- 変更履歴が残る

**デメリット:**
- 手動でコミット・プッシュが必要

**使い方:**
```bash
./sync_setup.sh
```

詳細は実行後に表示されます。

---

### 方法2: Rsync + SSH

**メリット:**
- 高速
- 細かい制御が可能

**デメリット:**
- 同じネットワーク内でのみ使用可能
- SSHサーバーのセットアップが必要

**使い方:**
```bash
./rsync_sync.sh
```

対話形式でセットアップできます。

---

### 方法3: Syncthing 【最も簡単】

**メリット:**
- 自動同期（何もしなくてOK）
- GUI管理画面あり
- インターネット経由でも同期可能

**デメリット:**
- 初回セットアップが少し複雑
- バックグラウンドで常に動く

**使い方:**
詳細は `syncthing_setup.md` を参照してください。

---

## 推奨事項

1. **開発用プロジェクト:** Git（方法1）
2. **簡単に同期したい:** Syncthing（方法3）
3. **一時的な同期:** Rsync（方法2）

## 複数の方法を併用することも可能！

例：
- メインはSyncthingで自動同期
- 重要なプロジェクトはGitでバージョン管理
