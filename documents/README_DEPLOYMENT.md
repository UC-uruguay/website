# 茶Bar 〜茶婆場〜 ウェブサイト デプロイメントガイド

## ファイル構成

以下のファイルがウェブサイトに必要です：

- `index.html` - メインのウェブサイトファイル（完全なレスポンシブデザイン）
- `uc.jpg` - UCさんのメンバー写真
- `haruhi.jpg` - はるきちさんのメンバー写真  
- `na.jpg` - なっちゃんさんのメンバー写真
- `ko.jpg` - こまっちゃんさんのメンバー写真

## アップロード方法

### 方法1: ファイルマネージャー（推奨）
1. ホスティングサービスの管理画面にログイン
2. ファイルマネージャーまたはFTPでサイトのルートディレクトリに移動
3. 既存のファイル（特にindex.html）をバックアップ
4. 上記5つのファイルをルートディレクトリにアップロード
5. https://chaba-ba.jpn.org でサイトを確認

### 方法2: WordPress管理画面
1. WordPress管理画面にログイン
2. 外観 > テーマエディター または 固定ページで直接HTML編集
3. index.htmlの内容をコピーしてトップページに貼り付け
4. メディアライブラリに写真をアップロード

## トラブルシューティング

- **写真が表示されない**: 写真ファイルがindex.htmlと同じディレクトリにあることを確認
- **スタイルが崩れる**: ブラウザキャッシュをクリアして再読み込み
- **モバイルで表示が崩れる**: viewport設定が正しいか確認（index.htmlに含まれています）
